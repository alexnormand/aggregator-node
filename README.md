Silex, Goutte)

Build instructions
===================

    $ git clone git://github.com/alexnormand/aggregator-node.git

    $ cd aggregator-node/server

    $ npm intstall

    $ node server.js

 open [http://localhost:3000/](http://localhost:3000/) in your browser
